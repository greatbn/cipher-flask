### Cipher Flask

#### description
Một ứng dụng nho nhỏ viết bằng Flask bao gồm RestAPI và Frontend (dự định sẽ sử dụng AJAX)
Tổng hợp các mật mã được học trên lớp (KMA) qua môn cơ sở ATTT

#### require

- flask
- flask_bootstrap
- python 2.7

#### How to use

- Install requirements

```
pip install -r requirements

```
- Run application

```
python main.py
```

- Access your browser

```
http://localhost:5000
```
